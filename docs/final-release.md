# OptimusCtx Final Release

## Status

OptimusCtx is deprecated.

The project is ending because benchmark runs did not show enough improvement over baseline workflows across the intended product goals to justify continued investment.

## Outcome

- The product idea remained coherent in theory, but the measured outcomes were not strong enough in practice.
- The benchmark results did not support continuing active development as a product bet.
- The repository remains available so the implementation, release artifacts, and lessons learned stay inspectable.

## Final release policy

- `v1.4.0` is the final planned release.
- No new features are planned after `v1.4.0`.
- Existing distribution channels remain archive paths for the last release line rather than active growth channels.
- If you still need the project, prefer installing or upgrading directly to `v1.4.0`.

## Distribution note

The final archived release line remains documented through:

- GitHub Release archives
- Homebrew
- Scoop
- npm as `@niccrow/optimusctx`

Those channels now exist to preserve access to the final release rather than to represent an actively developed product.
