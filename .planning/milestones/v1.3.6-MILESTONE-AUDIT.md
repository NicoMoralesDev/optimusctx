# Milestone v1.3.6 Audit Report

**Milestone:** v1.3.6 — Release channel truth and workflow modernization  
**Audited:** 2026-03-20T20:05:58Z  
**Status:** passed

## Summary

This milestone closes the downstream publication truth gap exposed by the `v1.3.5` release run. Homebrew and Scoop publication now treat first generated files in empty downstream repos as real pending changes, and the workflow only reports `publication_status=published` after a real repo update. Truthful no-op reruns now surface as `publication_status=already_current`.

The milestone also modernizes the release lane away from the deprecated Node 20-era action versions that were still warning in the workflow summary. The release preflight remains green for `v1.3.6`, with all channels wired and repository secrets detected.
